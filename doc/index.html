Алгоритм SMOTE

Что такое SMOTE?

SMOTE (неправильное название Synthetic Minority Oversampling Technique) — это популярный алгоритм, используемый для решения проблемы несбалансированных наборов данных путем генерации синтетических образцов для меньшинства. Цель алгоритма — сбалансировать данные для улучшения производительности моделей машинного обучения.

Как работает SMOTE?

Алгоритм SMOTE создает новые синтетические образцы следующим образом:

  1. Выбирается образец и ее k ближайших соседей из этого класса.
  2. Среди соседей случайным образом выбирается второй образец.
  3. На отрезке между этими двумя образцами создается новый синтетический образец.

Гиперпараметры SMOTE

Основные гиперпараметры алгоритма SMOTE:

  • k_neighbors: Количество ближайших соседей, используемых для генерации синтетических образцов (по умолчанию — 5).
